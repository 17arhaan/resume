Snake AI

Python | ReactJS

A modern take on the classic Snake game, developed in Python using Pygame, with integrated gesture control via OpenCV, and a web-based version built in React. This project allows players to enjoy the game using keyboard, AI, or hand gestures for control.

  • Created a multi-mode gameplay experience with AI, manual, and gesture control options.
  • Developed a responsive React web version with an intuitive user interface.
  • Implemented gesture recognition to control snake movements via webcam in the Python version.
Libraries : OpenCV | Pygame
